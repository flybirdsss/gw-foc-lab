# GW-FOC-Lab 任务日志空表 / Task Log Template

> 项目：GW-FOC-Lab
> 用途：记录 TODO、任务进度、Deadline、实际推进日志、问题记录和实验索引
> 建议文件位置：`docs/task_log.md`

---

## 1. 状态说明


| 状态     | 含义                           |
| ---------- | -------------------------------- |
| TODO     | 已确定要做，但还没开始         |
| DOING    | 正在做                         |
| BLOCKED  | 被问题卡住，需要讨论或等待条件 |
| DONE     | 已完成，并且有输出物           |
| PAUSED   | 暂时搁置，后续可能继续         |
| CANCELED | 确认不再做                     |

---

## 2. 任务类型说明


| 类型 | 说明                           |
| ------ | -------------------------------- |
| FW   | Firmware，嵌入式/FOC/MCU 相关  |
| HOST | 上位机/Python/Linux/可视化相关 |
| HW   | 硬件/电机/驱动板/接线相关      |
| DOC  | 文档/README/理论整理相关       |
| EXP  | 实验/数据采集/曲线分析相关     |
| TOOL | 工具脚本/数据处理/自动化相关   |
| MEET | 讨论/例会/方案决策相关         |
| BUG  | 问题定位/修复相关              |

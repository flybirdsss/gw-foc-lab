## 1. 状态说明

| 状态     | 含义                           |
| -------- | ------------------------------ |
| TODO     | 已确定要做，但还没开始         |
| DOING    | 正在做                         |
| BLOCKED  | 被问题卡住，需要讨论或等待条件 |
| DONE     | 已完成，并且有输出物           |
| PAUSED   | 暂时搁置，后续可能继续         |
| CANCELED | 确认不再做                     |

## 2. 任务类型说明

| 类型 | 说明                           |
| ---- | ------------------------------ |
| FW   | Firmware，嵌入式/FOC/MCU 相关  |

| DOC  | 文档/README/理论整理相关       |
| EXP  | 实验/数据采集/曲线分析相关     |

| MEET | 讨论/例会/方案决策相关         |
